### 赛前选择植物 🎯

点击关卡后进入 **植物选择界面**：

- 从全部 **11 种植物** 中挑选出战阵容

## 💀 僵尸博士 Boss 战

关卡 4 第 8 波登场，位于中间行。

| 属性 | V2 | V3 |
|------|----|----|
| 血量 | 4000 HP | **8000 HP** |
| 移动速度 | 0.07 | 0.07 |
| 体型 | 2.2× | 2.2× |
| 啃食伤害 | 3.0 | 3.0 |
| 技能 | 每 ~6s 召小僵尸 | 每 ~6s 召小僵尸 |
| 重做 | — | ✅ 召唤时红框闪烁提示 |

### 外观特征
- 灰色机械装甲身躯 + 红色管线
- 暴露的粉色大脑 + 血红双眼
- 机械爪子双臂 + 可伸缩手臂
- 头顶实时显示血条 + 数值
- 召唤前闪烁 `⚠ 召唤中...`

### 战斗策略
1. **火炬树桩 + 豌豆射手**：火焰豌豆对博士造成翻倍伤害
2. **大嘴花**：可以快速清理博士召唤的小僵尸
3. **坚果墙**：前排拖延，为输出争取时间
4. **三管射手**：多行覆盖，高效输出
5. **寒冰射手**：减速博士，降低推进速度
6. **樱桃炸弹**：小僵尸群密集时清场

| 操作 | 方式 |
|------|------|
| 选择关卡 | 点击关卡卡片 |
| 选择出战植物 | 点击植物卡片 / 数字键，`Enter` 确认 |

### 选植物界面

| 按键 | 功能 |
|------|------|
| `1`~`0` | 选择/取消对应顺序的植物 |
| `Enter` | 确认选择开始战斗 |

| 按键 | 植物 |

## ⚙️ 游戏机制

### 阳光经济
- 初始阳光：关卡 1~4 分别为 200/150/250/300
- 天空阳光：每 5 秒自动掉落
- 向日葵产出：每 ~4.7s 一颗 25 阳光
- 波次奖励：每波清完 +50
- 铲除返还：费用 50%
- 自动收集：落地 3 秒未点击自动收

### 火炬树桩 — 火焰豌豆机制
- 当豌豆射手的 **普通豌豆** 穿过火炬树桩所在格
- 豌豆变为 **火焰豌豆**：**伤害翻倍** + 灼烧效果
- 火焰豌豆带有橙色光晕特效
- 寒冰射手的冰豌豆**不会**被火炬树桩转化

### 大嘴花 — 吞噬机制
- 当僵尸进入大嘴花前方范围时，**一口吞掉**
- 吞噬无视僵尸血量（Boss 除外）
- 吞噬后进入 **8 秒消化期**，期间无法攻击
- 消化期大嘴花呈现咀嚼动画

### 撑杆跳僵尸 — 跳跃机制
- 移动速度快（0.30）
- 遇到第一个植物时 **撑杆跳过**
- 跳跃过程中对植物造成 60 伤害
- 跳过后失去撑杆，速度降低（0.13）

### 冰车僵尸 — 冰霜机制
- 高血量（500），移动慢
- **免疫减速效果**（寒冰射手无效）
- 身后留下冰尾迹（纯视觉特效）
- 身体周围有冰霜光环

### 割草机
每行左侧一台，僵尸接近时自动启动碾压。对 Boss 造成 500 伤害并减速。
