# 🌻 植物大战僵尸 V1 — 经典无尽模式

纯 HTML5 Canvas + JavaScript，单文件即玩。V1 是基础版，提供 **无尽波次** 挑战。

## 🚀 快速开始

```
双击 index.html
```

无需安装、无需服务器，浏览器打开即玩。

## 🎮 操作说明

| 操作 | 方式 |
|------|------|
| 选择植物 | 点击顶部植物卡片，或键盘 `1` `2` `3` `4` `5` |
| 放置植物 | 选中植物后点击草坪格子 |
| 收集阳光 | 点击阳光（3 秒未收会自动收集） |
| 铲除植物 | 点击 🔧 铲子按钮，再点击要铲的植物 |
| 取消选择 | 按 `ESC` 或鼠标右键 |
| 重新开始 | 游戏结束后点击「再来一局」 |

## ⚙️ 游戏机制

### 阳光经济
- **初始阳光**：200
- **天空阳光**：每 5 秒自动掉落一颗（25 阳光）
- **向日葵产出**：每约 4.7 秒产生一颗（25 阳光）
- **波次奖励**：每波清完奖励 50 阳光
- **铲除返还**：铲除植物返还费用的 50%

### 波次系统
- 第 1 波：4 只僵尸，生成间隔 5 秒
- 每波增加 1 只僵尸，波间休息 6 秒
- 僵尸类型随波次逐渐多样化
- 第 3 波开始可能出现铁桶僵尸

### 割草机
- 每行左侧有一台割草机，僵尸接近时自动启动
- 碾压该行所有僵尸后消失

### 胜负条件
- 💀 **失败**：任意僵尸到达左侧红线
- 🎉 **胜利**：无尽模式，挑战更高波次

## 🎨 技术实现

- **渲染**：HTML5 Canvas 2D
- **游戏循环**：`requestAnimationFrame` 60fps
- **碰撞检测**：基于坐标距离
- **粒子系统**：爆炸、击中、铲除等特效
- **单文件**：全部代码在 `index.html` 中
