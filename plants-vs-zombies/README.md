## 📜 版本进化史

### V1 — 经典无尽 🌻
- 5 种基础植物：向日葵、豌豆射手、坚果墙、樱桃炸弹、寒冰射手
- 4 种僵尸：普通、路障、铁桶、旗手
- 无限波次，难度递增
- 阳光经济、割草机、波次系统

### V2 — 关卡模式 🗺️
- 3 个独立关卡 + 星空选关画面
- 新增 2 种植物：**双发射手** 🔥、**土豆地雷** 💣
- 新增 **僵尸博士 Boss**（4000 HP，召唤小僵尸）
- 关卡氛围随关卡变化（草地→夜晚→暗红）

### V3 — 豪华版 💎
- 赛前选择出战植物系统
- 新增 4 种植物：**火炬树桩**（点燃豌豆 2 倍伤害）、**大嘴花**（一口吞）、**三管射手**（三行射击）、**小喷菇**（免费）
- 新增 2 种僵尸：**撑杆跳僵尸**（跳过植物）、**冰车僵尸**（免疫减速）
- 新增第 4 关：**终极对决**（Boss 战，博士 8000 HP）
- 粒子数量上限优化

### V4 — 豪华增强版 ⚡
- 新增 3 种植物：**倭瓜** 🎃（坐扁）、**辣椒** 🌶️（整行清屏）、**地刺** 🌵（持续伤害）
- 新增 1 种僵尸：**气球僵尸** 🎈（浮空越过植物）
- 新增第 5 关：**屋顶之战** 🏠（黄昏砖纹场景）
- **Delta Time** 帧率稳定技术（60Hz/144Hz 表现一致）
- 卡片冷却数字显示
- 选关左右拉条（5 关翻页 + 指示点）
- 气球僵尸互动优化（免疫地刺、大嘴花、割草机）

## 🎮 通用操作

| 操作 | 方式 |
|------|------|
| 选择植物 | 点击顶部卡片 / 键盘 `1`~`0` |
| 放置植物 | 选中后点击草坪格子 |
| 收集阳光 | 点击阳光 / 3 秒自动收集 |
| 铲除植物 | 点击 🔧 → 点击目标植物（返还 50% 阳光） |
| 取消选择 | `ESC` / 鼠标右键 |
| 翻页选关 | 点击 ◀ ▶ 箭头 / 键盘 `←` `→` |

## ⚙️ 技术特色

- **渲染**：HTML5 Canvas 2D
- **游戏循环**：`requestAnimationFrame` + Delta Time 固定步长（V4）
- **碰撞检测**：基于坐标距离
- **粒子系统**：爆炸、击中、铲除等特效
- **单文件架构**：每个版本独立一个 HTML，零依赖
